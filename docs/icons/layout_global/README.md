# Ikon Layout Global (Sidebar, Header, Profile Dropdown)

Folder ini berisi informasi mengenai ikon-ikon global yang termuat di berkas `layouts.admin` dan tampil pada seluruh halaman admin/konselor.

## Tabler Font Icons (`ti ti-*`)
Ikon-ikon berikut dimuat dari pustaka font eksternal `tabler-icons.min.css`:

### 1. Sidebar Navigation Menu
*   **Dashboard**: `ti ti-home` (Ikon Rumah)
*   **Penjadwalan**: `ti ti-calendar-event` (Ikon Kalender & Event)
*   **Chat**: `ti ti-message-circle` (Ikon Balon Dialog)
*   **Sesi Konseling**: `ti ti-stethoscope` (Ikon Stetoskop)
*   **Laporan Konseling**: `ti ti-file-report` (Ikon Dokumen Laporan)
*   **Edukasi**: `ti ti-book` (Ikon Buku)

### 2. Header Actions
*   **Toggle Sidebar**: `ti ti-menu-2` (Ikon Garis Tiga Hamburger)
*   **Web Push Notification**: `ti ti-bell` (Ikon Lonceng Notifikasi)

### 3. Profile & System Dropdown
*   **Profil Saya**: `ti ti-user` (Ikon Siluet Pengguna)
*   **Keluar / Logout**: `ti ti-logout` (Ikon Pintu Keluar)
*   **Arrow Trigger**: `ti ti-chevron-up` (Ikon Siku Panah Atas)

### 4. Modal Sukses Global
*   **Notifikasi Berhasil**: `ti ti-circle-check` (Centang dalam lingkaran)
*   **Tanda Centang List**: `ti ti-check` (Centang biasa)
